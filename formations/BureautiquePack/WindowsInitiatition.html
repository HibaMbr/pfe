Windows Initiation


Public concerné :
  Tout public.

Pré requis :
  Pas de prérequis spécifique.

L'objectif :
  • Etre à l'aise avec l'environnement de Windows.
  • Savoir gérer ses dossiers et fichiers efficacement (sauvegarde, recherche).
  • Utiliser les applications de base (Calculatrice, pense bête).

Programme de cours Windows Initiation:

  • Présentation de Windows.
  • Le matériel.
  • Les supports de sauvegarde.
  • L'environnement et outil virtuel.
  • Les outils de la fenêtre Windows.
  • L'arborescence de Windows.
  • Utilisation d'un support de sauvegarde.
